# FIFA World Cup 2026 Monte Carlo Simulator

## Overview

Interactive Monte Carlo simulator for FIFA World Cup 2026.

Features:

- Elo ratings
- Poisson goal model
- Group stage simulation
- Best third-place qualification
- Round of 32
- Round of 16
- Quarter-finals
- Semi-finals
- Final
- Champion probability
- Export CSV
- Export JSON
- PWA support
- GitHub Pages deployment

---

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript
- Chart.js
- Web Workers
- Service Workers

---

## Simulation Engine

Uses:

- Elo Ratings
- Poisson Goal Distribution
- Monte Carlo Sampling

Available iterations:

- 10,000
- 100,000
- 1,000,000
